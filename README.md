**Objetivo desta atividade**
Utilizando o Visual Studio, a linguagem C# e o aplicativo de console, crie
um jogo onde dois jogadores se alternam atacando. O combate continua até
que a saúde de um personagem chegue a zero, aplicando os
conhecimentos de POO (Programação Orientada a Objetos).

**● Regras:**
○ Mostrar a quantidade de dano que o jogador causou e com
quanta vida o outro jogador ficou;
○ Imprimir qual jogador venceu;
○ Deve conter no mínimo dois personagens (Mago e Guerreiro);
○ Deve conter dois atributos principais (ATAQUE e DEFESA);
○ Deve conter uma Interface chamada Personagem.

**Organização**
Essa atividade deve ser feita individualmente.
